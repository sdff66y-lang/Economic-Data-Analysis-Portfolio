# 经济数据分析项目组合

本仓库主要包含三个层次的分析报告：

- [中国制造业成本与经济周期分析](reports/China_Report.md)
- [中美制造业成本对比分析](reports/China_US_Report.md)
- [中美欧日韩五国制造业成本、经济周期与汇率波动对比分析](reports/Five_Countries_Report.md)

技术栈：Python + Pandas + Statsmodels + FRED API  
核心发现：中国制造业成本对工业生产的影响存在显著滞后2期效应，汇率贬值在中国和韩国对PPI影响较为明显。

项目包含代码、数据、图表和详细报告。
